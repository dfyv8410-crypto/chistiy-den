Политика конфиденциальности

Приложение «Чистый день»

1. Какие данные собираются

Приложение не собирает, не хранит и не передаёт персональные данные пользователей третьим лицам. Все данные (дата начала отсчёта, дела, служения, настройки) хранятся локально на устройстве пользователя и не покидают его пределы.

2. Сторонние сервисы

Приложение может загружать ежедневные медитации и расписание групп с сайта na-russia.org через общедоступные API и прокси-сервисы (corsproxy.io, allorigins.win, codetabs.com). Эти запросы не содержат персональной информации пользователя.

3. Уведомления

Приложение может запрашивать разрешение на отправку push-уведомлений для напоминаний о плане на день и служениях. Разрешение запрашивается только после явного действия пользователя. Уведомления обрабатываются локально и не передаются на внешние серверы.

4. Файлы cookie

Приложение не использует файлы cookie и не отслеживает действия пользователя.

5. Передача данных третьим лицам

Приложение не передаёт никакие данные третьим лицам. Все данные остаются исключительно на устройстве пользователя.
