# Чистый день

PWA-счётчик трезвости с ежедневными размышлениями «Только сегодня», расписанием групп АН, планом на день, системой медалей и списком служений.

## Возможности

- **Счётчик трезвости** — дни, недели, месяцы и часы с момента выбранной даты
- **Система медалей** — цветные медали от Белой (1 день) до Фиолетовой (3+ года) с прогресс-баром к следующей
- **Ежедневник «Только сегодня»** — ежедневные размышления сообщества Анонимные Наркоманы (автозагрузка с na-russia.org через 3 CORS-прокси + 31 встроенная медитация как fallback)
- **Расписание групп** — собрания АН в Москве и МО на сегодня с разбивкой по времени суток (API na-russia.org)
- **План на день** — список дел с отметкой выполнения и ежедневным напоминанием (через Notification API)
- **Медали** — традиционная система цветных медалей с визуальным прогрессом
- **SOS** — быстрый доступ к телефонам доверия и экстренным номерам
- **Служения** — календарь служений с еженедельными напоминаниями
- **Две темы** — светлая и синяя, переключение в настройках
- **PWA** — устанавливается на главный экран, работает офлайн через Service Worker
- **Настройки** — дата отсчёта, тема, поддержка проекта, документы (пользовательское соглашение, политика конфиденциальности, отказ от ответственности)

## Установка

### Быстрый старт

```bash
git clone https://github.com/dfyv8410-crypto/chistiy-den.git
cd chistiy-den
# Откройте index.html в браузере
```

### Установка на телефон (PWA)

1. Откройте приложение в Chrome на Android
2. Нажмите «Добавить на главный экран»
3. Приложение установится как иконка на рабочем столе и будет работать офлайн

### Публикация в RuStore

1. Опубликуйте сайт на GitHub Pages: Settings → Pages → main/root
2. Откройте URL в [PWABuilder](https://pwabuilder.com) → скачайте AAB
3. Загрузите AAB в RuStore Console с иконкой 512×512 и скриншотами

## Дизайн

- Flat design: без градиентов, теней, размытий — только сплошные цвета
- Основной цвет: `#1A3E6B` (тёмно-синий), фон: `#FFFFFF`, акцент: `#C8963E` (золото)
- Иконки: SVG на общей геометрии 512×512 (круг с кольцом → шестиугольник → внутренний круг → монограмма NA)
- Все элементы — Material-стиль с скруглениями 10–24px

## Настройка

**Цвета медалей и пороговые дни** — массив `MILESTONES` в `index.html`:

```js
const MILESTONES = [
  { days:0,   name:'Белая',     hex:'#F0EDE8', desc:'Первый день или готовность начать заново' },
  { days:30,  name:'Оранжевая', hex:'#E8924A', desc:'30 дней' },
  // ...
];
```

**Цветовые переменные** — CSS-переменные в `:root` (светлая тема) и `[data-theme="blue"]` (синяя).

## Приватность

Все данные хранятся локально в localStorage и IndexedDB. Приложение не собирает и не передаёт личные данные. Единственные внешние запросы — загрузка медитаций и расписания групп с публичного API na-russia.org.

## Технологии

- Чистый HTML/CSS/JS — без фреймворков и зависимостей
- Google Fonts (Fraunces + Manrope)
- Service Worker для кэширования и офлайн-режима
- IndexedDB + localStorage для хранения данных
- SVG-графика (все иконки)
