# Чистый день

PWA-счётчик трезвости с ежедневными размышлениями «Только сегодня», расписанием групп АН, планом на день, системой медалей и списком служений.

## Возможности

- **Счётчик трезвости** — дни, недели, месяцы и часы с момента выбранной даты
- **Система медалей** — цветные жетоны от белого (1 день) до фиолетового (3+ года) с прогресс-баром к следующей
- **Ежедневник «Только сегодня»** — ежедневные размышления сообщества Анонимные Наркоманы (с автоматической загрузкой с na-russia.org и локальным fallback)
- **Расписание групп** — собрания АН в Москве и МО на сегодня с разбивкой по времени суток (данные из официального API na-russia.org)
- **План на день** — список дел с отметкой выполнения и ежедневным напоминанием
- **Служения** — календарь служений с напоминаниями по дням недели
- **SOS** — быстрый доступ к телефонам доверия и экстренным номерам
- **Две темы** — светлая и синяя, переключение в настройках
- **PWA** — устанавливается на главный экран, работает офлайн через Service Worker

## Установка

Приложение — это статический сайт (HTML + CSS + JS). Достаточно открыть `index.html` в браузере.

### Быстрый старт

```bash
git clone https://github.com/dfyv8410-crypto/chistiy-den.git
cd zheton-dnya
# Откройте index.html в браузере
```

### Установка на телефон (PWA)

1. Откройте приложение в Chrome на Android
2. Нажмите «Добавить на главный экран» (или дождитесь подсказки)
3. Приложение установится как иконка на рабочем столе и будет работать офлайн

### Сборка APK (опционально)

1. Залейте папку на любой хостинг (Netlify, GitHub Pages, Vercel)
2. Откройте ссылку в [PWABuilder](https://pwabuilder.com)
3. Скачайте готовый APK для Android

## Настройка

**Цвета медалей и пороговые дни** — массив `MILESTONES` в `index.html`:

```js
const MILESTONES = [
  { days:0,   name:'Белая',   hex:'#F0EDE8', desc:'Первый день' },
  { days:30,  name:'Оранжевая', hex:'#E8924A', desc:'30 дней' },
  // ...
];
```

**Цветовые переменные** — CSS-переменные в `:root` (светлая тема) и `[data-theme="blue"]` (синяя).

## Приватность

Все данные (дата отсчёта, список дел, служения) хранятся локально в localStorage. Приложение не собирает и не передаёт личные данные. Единственные внешние запросы — загрузка медитаций и расписания групп с публичного API na-russia.org.

## Технологии

- Чистый HTML/CSS/JS — без фреймворков и зависимостей
- Google Fonts (Fraunces + Manrope)
- Service Worker для кэширования и офлайн-режима
- LocalStorage для хранения данных пользователя
